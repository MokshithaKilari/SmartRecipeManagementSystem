# Smart Recipe Management and Recommendation System
This web application was created by me as part of project work for DBMS course.
A web-based application that allows users to manage and explore recipes efficiently. 
The system provides features such as user registration, login, recipe management, and category-based recipe recommendations. 
Users can browse recipes based on Veg, Non-Veg, and Dessert categories while viewing ingredients, cooking instructions, cooking time, and difficulty level.

#Features

* User Registration and Login
* Secure Authentication
* Add New Recipes
* View Recipes
* Edit Recipes
* Delete Recipes
* Category-Based Recommendations
* Veg, Non-Veg, and Dessert Filtering
* Responsive User Interface
* Online Hosting Support

# Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap
* InfinityFree Hosting

# Project Objective

The objective of this project is to provide a simple and user-friendly platform for managing recipes and
recommending suitable dishes based on user-selected food categories.
